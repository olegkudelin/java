Микро описание программы

Описание
  Считает число слов в файле
  Если при запуске не было ничего передано, кроме имени файла, то возвращает полное число слов
  Если при запуске была указана строка, то считает только те слова, в которых встречатся указанная строка.
  Дополнительно можно указать следует ли учитывать регистр, если подсчитывается определенное слово.

  Состоит из 2-x основных классов:
   WordCounter - отвечает за подсчет слов в строке
   SearchOptions - отвечает за параметры поиска

   Точка входа в программу находится в WordCounter. Метод main принимает входные параметры коммандной строки, передает их SearchOptions, который парсит переданные параметры и формирует условия поиска. Далее происходит создание WordCounter с нужными условиями поиска. После чего открывается файл и начинается построчное чтение. Каждая строка передается WordCounter для подсчета количества слов, удовлетворяющих критериям. В завершении результаты подсчета каждой строки суммируется и выдается результат.

Сборка
   gradlew build
 собранный файл будет в build/libs

Запуск
 -Подсчет всех слов в файле: 
     java -jar simple-word-counting-1.0.jar file_name
   где file_name - имя файла с абсолютным путем

 -Подсчет слов, содержащих строку:
     java -jar simple-word-counting-1.0.jar -pattern substring file_name
   где substring - подстрока слова, которое будет учитываться при подсчете
   чтобы не учитывать регистр, необходимо добавить флаг -i

 -полный шаблон для запуска:
     java -jar build/libs/simple-word-counting-1.0.jar [-i] [-pattern word_pattern] file_name
